Made For Those Sick of Looking at HTML

Features

Clarity

Many tag and attribute names have been changed to make their purpose more clear. Several examples of this include: the div tag being changed to Container, the a tag being changed to Link, the p being changed to Text, and the href attribute being changed to destination.

Readability

Gossamer uses more verbose language as opposed to shortened language in order to increase readability. For example, instead of using src to specify an image's location, you use source.

Easier on the Eyes

Looking at a wall of HTML is not the most pleasant feeling. Gossamer forces tags to be spaced out more, which makes the code less intimidating. Gossamer also makes web code more C or Java like in style, making it feel more familiar to novice programmers.
